Aria

Con Accessible Rich Internet Applications si intendono un insieme di ruoli e attributi che definiscono meccanismi per rendere contenuti web e applicazioni web più accessibili a utenti con disabilità

La prima regola dell'uso degli ARIA è che se esiste un componente nativamente supportato dai browser è preferibile a qualsiasi implementazione personalizzata, le altre si potrebbero riassumere con Nessun aria è meglio di cattivi aria.
